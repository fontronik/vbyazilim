Eğitim ve Danışmanlık Hizmetleri

2013 yılında kurulan vb YAZILIM, eğitim ve danışmanlık hizmetlerinin yanı sıra, yazılım geliştiricileri için konferanslar, teknoloji firmaları adına hackathon’lar için danışmanlık servisi sağlamıştır.

2015 Mayıs ayından beri kişisel ya da kurumsal olmak üzere, hafta sonu eğitimlerimiz düzenli olarak devam ediyor. Modern programlama dilleri, yazılım geliştirme araçları, yazılım geliştirme metodolojileri ve benzeri konularda eğitim hizmetlerimiz bulunuyor.

Eğitim verdiğimiz kurumlar arasında Hürriyet WEB ekibi, Biges Güvenli Hayat Teknolojileri A.Ş bulunuyor.

Uzmanlık alanımız olan Python ve Ruby dillerinde geliştirilmiş ya da geliştirilecek yazılım projeleri için sağladığımız hizmetler;

  • Analiz ve süreç planlama
  • Süreç yönetimi
  • Altyapı analiz çalışması
  • Altyapı planlama
  • Veri teknolojileri seçimi
  • Uygulama göçü
  • Proje göçü
  • Platform göçü
  • Bakım / onarım servisileri
  • DevOps
  • Teknoloji seçimi
  • Bütçe, planlama ve ekip kurulumu
  • Kaynak kod yönetimi
  • Code Review ve Code Reliability hizmeti

Eğitim konuları;

git bash python django ruby regex macOS homebrew rubygems pypi tdd database

Eğitimlerimizden, bültenimizden, hizmetlerimizden haberdar olmak için bültenimize üye olabilirsiniz!
